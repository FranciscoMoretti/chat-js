---
title: add
description: Add a component or feature to an existing ChatJS project
---

```bash
npx @chat-js/cli@latest add [components...] [options]
```

## Arguments

| Argument       | Description                          |
| -------------- | ------------------------------------ |
| `componentsâ€¦`  | One or more component names to add.  |

## Options

| Flag              | Default | Description                                            |
| ----------------- | ------- | ------------------------------------------------------ |
| `-y, --yes`       | `false` | Skip confirmation prompt.                              |
| `-c, --cwd <cwd>` | `cwd`   | The working directory (defaults to current directory). |

## Status

<Note>
  The `add` command is not yet implemented. It is reserved for a future release that will let you add features, auth providers, and gateways to existing projects without re-scaffolding.
</Note>
